    Works

    • Falling Dot Hunter

      마우스 이벤트 핸들링과 비동기 타이머 구조의 에임 트레이닝 웹 콘텐츠

      • JavaScript
      • HTML5 Canvas
      GitHubPlay

      Technical Challenges

      • 입력 파이프라인 최적화: 마우스 궤적 선형 보간 알고리즘을 구현하여 주사율 병목으로 인한 포인터 렉을 해결하고, 캔버스 내 십자선의 추적 반응성을 극대화함.
      • 비동기 이벤트 루프 스케줄링: 난이도에 따른 가속 매커니즘을 선형 제어하고, 브라우저 이벤트 루프 스케줄링을 연동하여 시스템 타임아웃 예외를 방지하는 타이머 사이클을 설계함.

    • Galaxy Defender

      상수의 모듈화 구조와 벡터 연산을 적용한 종스크롤 슈팅 웹게임

      • JavaScript (Vanilla JS)
      • HTML5 Canvas
      • Web Storage

      Technical Challenges

      • SSOT 구조: 모든 밸런싱 수치와 시스템 상수를 전역 상태 객체로 통합 관리하여 런타임 결합도를 낮추고 유지보수 생산성을 극대화함.
      • 방향 벡터 연산 최적화: 삼각함수와 피타고라스정리를 활용한 실시간 방향 벡터 연산으로 계산 비용을 최소화한 플레이어 조준 투사체 및 선형 보간 기반 유도 추적 시스템을 설계함.
      • 세션 수명주기 예외 처리: Page Visibility API와 런타임 이벤트를 동기화하여 유저 세션 이탈 시 발생하는 데이터 유실 및 렌더링 무한 루프를 원천 차단함.

    • MESSIAH

      순수 자바스크립트로 구현한 하이퍼포먼스 Top-down 로그라이크 서바이벌 게임

      • JavaScript (ES6+)
      • HTML5 Canvas API
      • Component-Based Architecture

      Technical Challenges

      • 공간 분할 및 오브젝트 풀링: 상용 엔진 없이 그리드 기반 공간 분할 충돌 필터링과 800개 규모의 오브젝트 풀링 시스템을 자체 구축하여 GC 오버헤드를 최소화하고 FPS를 안정적으로 방어함.
      • 확장형 FSM 설계: FSM 아키텍처를 기반으로 대시, 도약, 트리거형 범위 경고 등 5단계의 보스 AI 루틴을 예외 처리 버그 없이 안정적으로 결합함.

    • Cat Busters

      2D 레트로 스프라이트 연출과 3D 공간 물리 연산을 결합한 2.5D 탑다운 슈팅 게임

      • C#
      • Unity 3D Engine
      • Cinemachine
      • 3D Rigidbody Physics
      GitHubDownload

      Technical Challenges

      • 물리 충돌 및 동적 가속도 제어: 물리 컨텍스트 내에서 가속도 튕김이나 밀림 현상을 방지하기 위해 Kinematic 모드와 Rigidbody.MovePosition을 결합한 안정적인 이동 동역학을 구현함.
      • 가변 사거리 및 레이캐스팅 연산: 오브젝트별 고유 충돌 반경(Radius)을 프레임마다 가중 연산하여 유동적인 전투 거리를 산출하는 가변 사거리 알고리즘을 개발함.
      • 선형 보간 UI 애니메이션: 월드 스페이스의 3D 좌표를 뷰포트 스크린 좌표계로 오차 없이 변환하는 인터랙티브 렌더러를 구축함.

    • Otherside Quest

      2D 레트로 스프라이트 연출과 3D 공간 물리 연산을 결합한 2.5D 쿼터뷰 액션 RPG

      • C#
      • Unity 3D Engine
      • URP (Universal Render Pipeline)
      • 3D Rigidbody Physics

      Technical Challenges

      • 2.5D 그래픽스 파이프라인: 쿼터뷰 환경의 벽면 파고듦을 막는 카메라 동기화 빌보드 시스템을 구축함.
      • 렌더링 최적화 및 투영 가림: 카메라 절두체 컬링 시스템을 커스텀 설계하여 드로우콜을 절약함.
      • 입력 버퍼링 및 물리 FSM: 연속 공격의 조작감을 위해 연타 입력을 일시 보관하는 공격 입력 버퍼링 알고리즘을 구현함.
